Шаблон бенчмарка (с ними вы еще познакомитесь подробнее в модуле 06.05) выглядит 
следующим образом: 
func BenchmarkWithoutPool(b *testing.B) {
	for i := 0; i < b.N; i++ {
		for j := 0; j < 10000; j++ {
			b.StopTimer()
			putInt(i)
			b.StartTimer()
		}
	}
}
Запуск выполняется следующей командой: 
go test -bench=. bench_test.go
